# ビットコイン自動売買システム

## デモ
デモムービーです
https://shimotmk.com/Portfolio/movie/team2.mp4

## 説明
デモムービーでは過去30秒のビットコインの価格を正規分布と仮定し上位0.03%の場合は売りもしくは買いをするプログラムにしています<br>
ボリンジャーバンドを利用しています<br>
実際はもう少し範囲を伸ばしています<br>
IFDOCO注文も可能です

## 必要なもの
ビットフライヤーAPI
https://bitflyer.com/ja-jp/api

## 言語
* Ruby

## 環境
